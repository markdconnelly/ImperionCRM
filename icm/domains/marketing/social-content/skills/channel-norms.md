# Skill (Tier 3, workflow-local): channel-norms

Mark-editable per-network hard constraints for adapting one composition into each
channel's post. The shared *tone* lives in `domains/marketing/skills/brand-voice.md`;
this file is the *format* — lengths, link/hashtag rules, and what each network refuses.
Stages cite this; they never restate it. Values are starting defaults, tune freely.

| Channel | Length target | Links | Hashtags | Media | Notes |
|---|---|---|---|---|---|
| Facebook page post | 1–3 short paragraphs | inline OK | 0–2 | image/video strong | conversational; question or CTA to close |
| Instagram media | caption ≤ 2,200 chars, front-load first line | not clickable — "link in bio" | 3–8, end of caption | image/video required | visual-first; caption supports the image |
| Threads post | ≤ 500 chars | inline OK | sparingly | optional | dry, human, reply-friendly; no LinkedIn formality |

## Rules that bind every channel

- **One CTA** per post; one idea.
- **Adapt, don't duplicate** — a Threads post is not a repaste of the LinkedIn copy.
- **Alt text** on every image (accessibility).
- **No client identities, no confidential data** in public copy (BO-01 §4.6).
- **Frequency caps** are honoured — a routine post is to an *established* audience
  within the cap; anything beyond that is a blast (→ always-gate, stage 03).
- A channel without a live credential/token is **dormant**: stage the adaptation,
  say so, never present a dormant channel as published.
